# [Конфигурации] (#config)


## [Indention Options](#indention)

`set autoindent` "Новые строки наследуют отступ от предыдущих строк

`set expandtab` "Преобразование вкладок в пробелы.

`set filetype indent on` "Включить правила отступа в зависимости от типа файла

`set shiftround` "При смещении строк округлять отступ до ближайшего значения кратного параметру shiftwidth

`set shiftwidth=4` "При смещенении делать отступ в 4 пробела.

___

## [Search Options](#search)

`set hlsearch` "Включить выделение результатов поиска.

`set ignorecase` "Игнорировать регистр при поиске.

`set incsearch` "Инкрементированный поиск, показывать частичное совпадение.

`set smartcase` "Автоматический переключать поиск на чувствительный к регистру, когда поисковый запрос содержит заглавную букву.

___

## [Performance Options](#perf)

`set lazyredraw` "Не обновлять экран во премя выполнения макроса и скрипта. Параметр отрисовки текста.

`set display+=lastline` "Всегда стараться отображать последнюю строку абзаца.

`set encoding=utf-8` "Использовать кодировку поддерживающую юникод.

`set linebreak` "Избегать переноса строки в середине слова.

`set scrolloff=1` "Количество экранных строк над и под курсором.

`set sidescrolloff=5` "Количество столбцов экрана, которые нужно держать слева и справа от курсора.

`syntax enable` "Включить подсветку синтаксиса.

`set wrap` "Включить перенос строк.

___

## [User Interface Options](#user)

`set laststatus=2` "Всегда отображать строку состояния.

`set ruler` "Всегда показывать позицию курсора.

`set wildmenu` "Показывать все параметры вкладки командной строки в виде меню.

`set tabpagemax=50` "Максимальное количество страниц вкладок, которые могут быть открыты из командной строки.

`set colorscheme wombat256mod` "Изменить цветовую схему.

`set cursorline` "Выделить строку, которая в данный момент находится под курсором.

`set number` "Показывать номера строк на боковой панели.

`set mouse=a` "Включить мышь для прокрутки и изменения размера.

`set title` "Задайте заголовок окна, соотвествующий редактируемому файлу.

`set background=dark` "Используйте цвета, подходящие для темного фона.

___

## [Code Folding Options](@code)

`set foldmethod=indent` "Складывание на основе уровней отсупа.

`set foldnestmax=3`: "Сворачивать только до трех ууровней вложенности.

`set nofoldenable` "Отключить сворачивание по умолчанию.

___


## [Miscellaneous Options](#miscell)

`set autoread` "Автоматический перечитывать файлы, если они не были изменены внутри Vim.

`set backupdir=~/.cache/vim` "Каталог для хранения файлов резервных копий.

`set confirm` "Показывать диалоговое окно подтверждения при закрытии несохраненного файла.

`set dir=~/.cache/vim` "Каталог для хранения файлов подкачки.

`set hidden` "Скрыть файлы в фотновом режиме, а не скрывать их.

`set history=1000` "Увеличить лимит отмены.

`set noswapfile` "Отключить файлы подкачки.

`set nrformats-=octal` "Интерпритировать восьмеричное как десятичное при увеличении чисел.

`set shell` "Оболочка, используемая для выполнения команд.

`set spell` "Включить проверку орфографии.
